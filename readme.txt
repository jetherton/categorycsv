tested up to: 2.4
author: John Etherton

== Description ==
This plugin lets you have all of your data broking out into seperate columns in a CSV. No more trying to pull about the comma delimited list of categories in the usual CSV.



== Installation ==
1. Copy the entire /categorycsv/ directory into your /plugins/ directory.
2. Activate the plugin.
